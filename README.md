<h2>Configuration</h2>

Config - `addons/counterstrikesharp/configs/plugins/KVSK_X`
```json
{
  "TagServer": "[KvsK]",         //Глобальный тег
  "ShowHUD": 1,                  // Худ на кол-во игроков. 
  "SetHealt": 35,                // Установка HP в начале раунда 
  "ConfigVersion": 1
}
```
